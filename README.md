# Mini ERP Database Management System

This project demonstrates an ERP-style relational database built using MySQL.

## Features
- Normalized tables with primary and foreign keys
- SQL joins, subqueries, and aggregate functions
- Views for reporting
- Stored procedures for reusable business logic
- Indexes for performance optimization

## Tech Stack
- MySQL
- SQL

## How to Run
1. Execute `01_create_database.sql`
2. Execute `02_create_tables.sql`
3. Execute `03_insert_data.sql`
4. Execute remaining SQL files
